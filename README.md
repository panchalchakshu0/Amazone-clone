# Amazon Clone

A simple Amazon homepage clone created using HTML and CSS.  
This project focuses on practicing frontend skills by recreating the layout and design of Amazon.

## Live Demo

Netlify: (https://amazonnclone16.netlify.app)

## Features

- Navigation bar similar to Amazon
- Search bar design
- Product category boxes
- Footer section
- Responsive layout

## Tech Stack

- HTML
- CSS
- Font Awesome
